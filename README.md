# lemonico-api

This is a practice project about automatic code generation. 

## Description

You can use gradle tasks to generate a lot of CRUD code. For example Controller, Service, Entity, Dao and SQL file.
```
#gradle ⇒ gen tasks

genAll
genController
...
```
